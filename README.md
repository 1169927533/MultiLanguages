## MultiLanguages (support Android O)
- Support system language switching with it
- Support users to select language switching (restart will not be restored)

## 多语言（国际化，支持Android 8.0）
- 支持系统语言切换随之切换
- 支持用户选择语言切换（重启不会还原）
